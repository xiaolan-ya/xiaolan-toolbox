小蓝工具箱 Mac 端测试和打包教程 v16

一、准备
1. 确认 Mac 已安装 Node.js 24。
2. 把 Windows 桌面上的 xiaolan-mac-test-v16-mac-safe.zip 传到 Mac 桌面。
3. 打开 Mac 的“终端”。

二、解压并进入项目
注意：不要 cd 进入 .zip 文件，zip 需要先解压。

复制运行：

cd ~/Desktop
unzip xiaolan-mac-test-v16-mac-safe.zip
cd xiaolan-mac-test-v16

三、安装依赖

npm install

四、先测试

npm run test:mac

看到类似下面结果就表示测试通过：

"layoutIssueCount": 0
"clickIssueCount": 0
"runtimeErrorCount": 0

如果卡在 electron tools/ui-layout-smoke.js，先等 2 到 3 分钟。
如果还是不动，可以点一下弹出来的 Electron / 小蓝工具箱窗口，让它保持前台。

五、打 Mac 包

CSC_IDENTITY_AUTO_DISCOVERY=false npm run dist:mac:unsigned

注意：命令前面不要加 [，完整就是上面这一行。

六、找到生成文件

打包完成后文件在：

~/Desktop/xiaolan-mac-test-v16/dist/

一般会生成：

小蓝工具箱-1.3.511-b-arm64.dmg
小蓝工具箱-1.3.511-b-arm64-mac.zip

七、复制到桌面

cp dist/*.dmg ~/Desktop/
cp dist/*mac.zip ~/Desktop/

八、如果 npm 报找不到 package.json

说明当前终端路径不在项目文件夹里。
重新运行：

cd ~/Desktop/xiaolan-mac-test-v16

然后再运行：

npm run test:mac

九、如果提示 zip 不是目录

说明你运行了类似：

cd xiaolan-mac-test-v16-mac-safe.zip

这是错误的。正确做法是：

cd ~/Desktop
unzip xiaolan-mac-test-v16-mac-safe.zip
cd xiaolan-mac-test-v16

十、Windows 端文件

Windows 可直接运行版本在 Windows 桌面：

xiaolan-toolbox-fixed-v16.exe
